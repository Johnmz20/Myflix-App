myFlix Movies collection

Welcome to myFlix movie collection

we have informatin of many movies of each genres. you can create your personal account and make a list of your favorite movies!

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies

EndPoints:

Request URL HTTP Method Response
Return a list of movies to the user /movies GET a JSON object holding data about all movies
Return Movies data (Description, Genre, Director, Image URL) about a movie by title to the user /movies/title GET a JSON object holding data about a single movie containing the data above
Return data about Movies genre (description) by name/title /movies/genre/genreName GET JSON object holding data about genre
Return data about director of each movie (bio, birth year) by name /movies/directors/directorName GET JSON object holding data about director
Allow new users to register /users POST a text message indicating successfull registration of the account
Allow users to update their user info (username) /users/id PUT a text saying the info got updated
Allow users to add a movie to their list of favorites /users/id/movieTitle POST a text message that movie was added to favorite list
Allow users to remove a movie from their list of favorites /users/id/movieTitle DELETE text message that movie was removed from favorites list
Allow existing users to deregister /users/id DELETE a text message indicating successfull deletion of the user account
